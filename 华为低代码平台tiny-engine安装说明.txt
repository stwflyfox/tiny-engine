nvm use 20.10.0  
npm install -g pnpm
安装data-center:
    git 克隆 ： https://github.com/stwflyfox/tiny-engine-data-center.git , 
    进入克隆后的目录，
          npm install -g yarn
          yarn install --ignore-engines
          npm run build
安装 webservice:
    git 克隆： https://github.com/stwflyfox/tiny-engine-webservice.git , 
    进入克隆后的目录
          yarn install --ignore-engines
    设置windows环境变量：
         MYSQL_HOST  ： 60.204.187.130
         MYSQL_PORT  ： 3306
         MYSQL_PASSWORD  ：Akcw2019!@#
         MYSQL_DBNAME  ： tiny_engine
     注销windows
安装 tiny-engine:
    git 克隆：https://github.com/stwflyfox/tiny-engine.git , 
    进入克隆后的目录, 切换到 release 分支，
         运行 pnpm install
         运行 pnpm run build:plugin
         运行 pnpm run build:alpha
以上安装完成后，按序进入 data-center， webservice ， tiny-engine 各目录运行 dev 即可
